# Keyboard Shortcuts

Quick reference for all editor keyboard shortcuts.

## Viewport Tools

| Shortcut | Action |
|----------|--------|
| Shift+Q | Select tool |
| Shift+W | Move tool |
| Shift+R | Scale tool |
| Shift+E | Ruler tool |
| Escape | Clear ruler measurement |

## File Operations

| Shortcut | Action |
|----------|--------|
| Ctrl+N | New scene |
| Ctrl+S | Save scene |

## Edit Operations

| Shortcut | Action |
|----------|--------|
| Ctrl+D | Duplicate selected entity |

## Navigation

| Shortcut | Action |
|----------|--------|
| Ctrl+R | Reset camera position |

---

The Keyboard Shortcuts panel (accessible from the editor menu) shows all registered shortcuts and their current bindings.
