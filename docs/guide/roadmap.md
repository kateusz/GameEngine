# Roadmap

Features under active development or planned for future releases.

| Feature | Description | Status |
|---------|-------------|--------|
| Runtime UI System | Canvas, Button, Label, Panel components and layout system for in-game UI | Planned |
| Particle System | 2D particle emitters with velocity, color, and size over lifetime | Planned |
| Scene Hierarchy | Parent/child entity relationships with local transforms | Planned |
| Gamepad Support | Xbox/PlayStation controller input, analog sticks, vibration | Planned |
| 3D Model Loading | FBX format support via Assimp | Planned |
| Lighting System | Point lights, directional lights, and shadows for 2D and 3D | Planned |

This list reflects current priorities and may change.
